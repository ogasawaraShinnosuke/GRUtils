# GRUtils

## Get Started

### Gradle

``` gradle
allprojects {
    repositories {
        maven { url "https://jitpack.io" }
    }
}

dependencies {
        compile 'com.github.ogasawaraShinnosuke:GRUtils:0.0.1'
}
```

### Maven

``` xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
<dependency>
    <groupId>com.github.ogasawaraShinnosuke</groupId>
    <artifactId>GRUtils</artifactId>
    <version>0.0.1</version>
</dependency>
```
